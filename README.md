# QR Scanner Compose

A simple QR code scanner Android app built using **Jetpack Compose**.

## Features
-  Built with Jetpack Compose
-  Scan QR codes in real-time
-  Lightweight and fast
-  Minimal dependencies

## Tech Stack
- **Kotlin**
- **Jetpack Compose**
- **CameraX API**
- **Android ViewModel & LiveData**
